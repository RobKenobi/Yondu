git status : gives informations about the current changes in the repository

git add <file> : add untracked/modified <file> to the git (<file> is now tracked)
git add * : add all untracked/modified files to the git (some files may not be added with this method, always check with git status and add them manually)

git commit -m "<message>" : save the changes added to the git. The changes are saved locally (on your computer only)

git push : send the commited work online

git pull : download the changes from the online repository

git stash : save your work and go back to the last commited version
git stash apply : apply the last stashed work

git log : log of the last commits
